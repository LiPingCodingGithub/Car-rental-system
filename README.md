## 汽车租赁系统(web前台+web后台管理)  --  毕业设计项目
### 以下是web前端页面的首页

![show page](https://github.com/LiPingCodingGithub/Car-rental-system/blob/master/qichezulin/01.png) 


### web前台(功能简介)

- 汽车租赁下单
- 用户登录功能
- 用户注册功能
- 订单查询
- 汽车归还



### web后台(功能简介)

- 管理汽车信息
- 管理用户信息
- 管理商家管理员admin的信息
- 管理订单信息
- 登录、注册功能



#### 相关技术介绍

* web前端：HTML5、CSS3、原生javascript
* 后台程序：使用PHP技术
* 数据库：使用了关系型数据库Mysql
* web服务器：Apache服务器
